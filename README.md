# Top Eats
Find and review local restaurants.

A full-stack app that allows you to search based on food and location, and returns results from Yelp's business API.
Allows users to sign-up and review restaurants.

Created using create-react-app.

Uses MERN stack: React, Node, Express, MongoDB

## Dependencies
- react
- axios
- react-bootstrap (styling)
- react-router-dom (navigation)
- brcypt (user encryption)
- jsonwebtoken (authentication)
- react-bootstrap-star-rating (review's star rating) 
- react-toastify (notifications)
 
## APIs
Yelp Fusion: https://www.yelp.com/developers/documentation/v3/business_search
